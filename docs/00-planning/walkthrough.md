# 前端 UI Demo 运行指南

您现在可以查看 "小星伴" (Little Star) 的最新 UI 原型了！
这个 Demo 展示了我们定义的 **"Juicy & Gamified" (多汁且游戏化)** 视觉语言，包括核心的黄色品牌色、Q弹的按钮交互以及报告详情页。

## 🚀 快速开始

在终端中运行以下命令：

```bash
cd prototype/frontend-demo
npm run dev
```

然后打开浏览器访问显示的链接（通常是 `http://localhost:5173`）。

## ✨ 体验亮点

### 1. 🌟 首页 (Home)
- **Hero Card (核心卡片)**：注意那个巨大的黄色"查看今日报告"按钮，按下去有真实的**3D下陷感**。
- **状态胶囊**：顶部的设备电量动态条。
- **FAB 按钮**：右下角的录音按钮，点击会弹出"正在聆听"的模拟界面。

### 2. 📊 报告详情页 (Report Detail)
- **如何进入**：点击首页巨大的 Hero Card 上的按钮。
- **情绪图表**：注意它是用 Emoji 连接起来的动态图表。
- **💡 互动建议**：
    - 点击"互动建议"卡片（如"关于恐龙的兴趣"）。
    - 展开后，您会看到**对话剧本**。
    - 试着点击"试着这样聊"的气泡，感受交互样式。

### 3. 🎨 视觉细节
- **圆角**：所有卡片和按钮都使用了 `Rounded` 字体和超大圆角，极具亲和力。
- **颜色**：核心的 **Star Yellow (#FFC800)** 贯穿始终，配合 **Sky Blue** 和 **Magic Purple** 作为辅助。
