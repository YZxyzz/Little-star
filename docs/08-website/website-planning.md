# 小星伴官方网站规划

> **版本**：v1.0
> **创建日期**：2025-02-06
> **设计风格**：Moimoi Style (与APP一致)

---

## 一、网站定位

### 1.1 网站目标

| 目标层级 | 具体目标 | 成功指标 |
|---------|---------|---------|
| **品牌认知** | 建立"懂孩子的成长伙伴"品牌印象 | 品牌记忆度 |
| **产品理解** | 清晰传达产品核心价值 | 页面停留时间 >2min |
| **信任建立** | 消除隐私安全顾虑 | 跳出率 <40% |
| **转化驱动** | 引导预购/众筹/下载APP | 转化率 >3% |

### 1.2 目标用户

| 用户群体 | 特征 | 核心关注点 |
|---------|------|-----------|
| **主要** | 3-7岁孩子的妈妈，25-35岁 | 了解孩子、教育引导 |
| **次要** | 3-7岁孩子的爸爸 | 技术安全、产品功能 |
| **潜在** | 准父母、祖父母 | 送礼价值、使用难度 |

### 1.3 网站类型

**品牌官网 + 产品落地页**

- 不是电商网站（购买跳转众筹/淘宝）
- 不是内容网站（不需要大量文章）
- 核心是：讲好品牌故事 + 产品价值

### 2.4 布局原则

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         布局设计原则                                         │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   1. Unboxed (去盒化)                                                       │
│      - 不要把内容关在方形盒子里                                              │
│      - 使用不规则形状作为容器                                                │
│      - 内容与背景自然融合                                                    │
│                                                                             │
│   2. Overlapping (重叠)                                                     │
│      - 元素之间可以有轻微遮挡                                                │
│      - 制造前后景深感                                                        │
│      - IP角色可以"跳出"容器边界                                              │
│                                                                             │
│   3. Floating (悬浮)                                                        │
│      - 所有可交互元素感觉是"浮"在白纸上                                       │
│      - 使用柔和阴影增强悬浮感                                                │
│      - 按钮和卡片要有弹性质感                                                │
│                                                                             │
│   4. Organic Shapes (有机形状)                                              │
│      - 拒绝直线，拥抱曲线                                                    │
│      - 使用非对称圆角 (e.g., 60% 40% 30% 70%)                               │
│      - 背景装饰使用气泡/云朵形状                                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

```css
/* 次要按钮 - Ghost Pill */

  background: #FFFFFF;
  color: #000000;
  border-radius: 100px;
  padding: 16px 32px;
  font-weight: 700;
  box-shadow: 0 4px 20px rgba(0,0,0,0.1);
}

/* 强调按钮 - Pink Pill */
.btn-accent {
  background: #FF8FAB;
  color: #FFFFFF;
  border-radius: 100px;
  padding: 16px 32px;
  font-weight: 800;

```

#### 气泡容器
```css
/* 有机形状容器 */
.blob-container {
  border-radius: 60% 40% 30% 70% / 60% 30% 70% 40%;
  padding: 40px;
  background: #FF8FAB; /* 或其他气泡色 */
}
```

### 2.6 动画与交互

| 交互类型 | 动画效果 | 时长 | 缓动函数 |
|---------|---------|------|---------|
| 按钮悬停 | 放大1.05 + 阴影加深 | 0.2s | ease-out |
| 按钮点击 | 缩小0.95 | 0.1s | ease-in |
| 页面滚动 | 元素渐入 + 上移 | 0.4s | ease-out |
| 气泡装饰 | 缓慢漂浮 | 6-10s | ease-in-out |
| IP角色 | 微微晃动/眨眼 | 循环 | ease-in-out |
