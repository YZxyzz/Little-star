# 小星伴硬件项目开发费用估算 (MVP & Mass Production)

本文档基于"深圳供应链"标准，为您估算从**MVP验证**到**首批小量产**的整体投入成本。
费用主要分为两部分：
1.  **NRE (一次性工程费用)**：设计、打样、开模等"沉没成本"。
2.  **BOM (单台硬件成本)**：每生产一台设备的物料成本。

---

## 一、 NRE 一次性工程费用 (研发阶段)

这是在产品正式量产前，您需要投入的研发和设计费用。

| 项目 | 预估费用 (RMB) | 说明 |
| :--- | :--- | :--- |
| **1. 工业设计 (ID Design)** | **3,000 - 8,000** | 找自由设计师或淘宝设计工作室。设计产品的外观、尺寸、材质。输出渲染图。 |
| **2. 结构设计 (MD Design)** | **3,000 - 6,000** | 设计内部塑料壳体的 3D 图 (STP)，确保电路板和电池能塞进去，且卡扣能合上。 |
| **3. 毛绒公仔打样 (Plush Sample)** | **300 - 800 /次** | "打版费"。找毛绒玩具厂打样，通常需要修改 2-3 次才能达到理想效果。 **(首板非常重要)** |
| **4. 硬件手板 (Prototyping)** | **200 - 500** | 3D 打印或 CNC 加工内部塑料壳体，验证结构是否合理。 |
| **5. 硬件电路开发 (PCBA)** | **0 (自研)** 或 **3,000+** | 如果用模块搭建 MVP，费用为 0。如果找人画 PCB 专板，约 2000-5000 元设计费。 |
| **6. 软件/固件开发** | **0 (自研)** | 假设由您自己或团队内部开发。如果外包，简单的功能约 10k-20k。 |
| **NRE 总计预估** | **~7,000 - 18,000** | **准备约 2万元启动资金**用于前期的打样和设计验证是比较稳妥的。 |

> **💡 MVP 阶段省钱技巧**：
> 1.  **外观**：先不找专业 ID，直接用现成的毛绒公仔改制。
> 2.  **结构**：用 3D 打印机打印简单的盒子装电子件，直接塞进公仔里。
> 3.  **电路**：直接用淘宝买的模块 (ESP32板+4G板) 用导线焊接，**不画PCB**。
> **MVP 阶段仅需投入：毛绒打样费 (~500) + 购买模块费 (~500) ≈ 1000元即可启动！**

## 二、 MVP 阶段单台硬件成本 (积木式组装)

此阶段**不追求成本最低**，只追求**跑通流程**。我们直接采购淘宝现成的开发板模块。

| 模块 | 淘宝零售价 (RMB) | 备注 |
| :--- | :--- | :--- |
| **ESP32-S3 核心板** | 25.0 | 比如合宙或者安信可的开发板，带USB口 |
| **Air780E 4G开发板** | 39.0 | 带天线、SIM卡槽、USB口的完整板子 |
| **INMP441 麦克风模块** | 8.0 | 现成的引脚模块 |
| **MAX98357 功放模块** | 6.0 | 现成的模块 |
| **28mm 喇叭** | 3.0 | - |
| **600mAh 锂电池** | 15.0 | 零售价格 |
| **杜邦线/耗材** | 5.0 | - |
| **MVP 单机成本** | **~101 元** | **加上毛绒公仔成本，单台约 120-130元**。 |

> **结论**：MVP 阶段单台成本虽高 (约120元)，但无需开模费，非常适合做 5-10 台送给种子用户测试。

## 三、 量产阶段 BOM 成本估算 (1000台+)

当 MVP 验证成功，需要将电路集成到一张 PCB 上，并批量采购元器件时，成本会大幅下降。
*(参考之前的硬件规格书，针对 5cm 吊坠版调整)*

| 项目 | 批量单价 (RMB) |
| :--- | :--- |
| **PCBA (所有电子料+贴片)** | **~65 - 75** | (ESP32芯片13元 + 4G模组25元 + 电池12元 + 其他阻容感连接器) |
| **结构件 (模具摊销+注塑)** | **~5 - 8** | 内部塑料保护壳 (需开钢模，约2-3万元模具费，摊销后单价低) |
| **毛绒外壳** | **~10 - 15** | 批量生产成本较低 |
| **包装/组装/测试** | **~8 - 10** | 人工成本 |
| **量产综合成本** | **~90 - 110 元** | *注：量产 1000 台量级，供应链议价能力有限* |

## 四、 软件服务成本 (持续性投入)

您特别提到的**实时翻译**和**内容总结**，这部分主要是**云端 API 费用**。

1.  **语音转文字 (STT/ASR)**:
    *   **阿里/百度/火山引擎**: 约 0.003元/60秒 (极便宜)。
    *   **OpenAI Whisper**: 效果最好，但成本稍高，或者自己部署服务器 (GPU成本)。
2.  **大模型总结 (LLM)**:
    *   **DeepSeek / Kimi / 豆包**: 输入+输出约 10-20元 / 百万 tokens。
    *   *估算*：每天录音 2小时 → 转文字约 2万字 → 总结消耗 1万 tokens → **每天每人仅需 ~0.1 - 0.2 元**。
    *   **结论**：软件服务成本非常低，完全可以覆盖在 39.9元/年的订阅费中。

## 五、 建议执行路线

1.  **阶段一：验证 MVP (本月)**
    *   **投入**：~1000元。
    *   **行动**：淘宝买模块，自己焊接；买现成的星星公仔，把棉花掏出来塞进去。
    *   **目标**：跑通 [录音 -> 4G上传 -> 生成总结] 的全流程。验证**"内容总结准确度"**。

2.  **阶段二：原型机 (下个月)**
    *   **投入**：~5000元。
    *   **行动**：找人画简单的 PCB (把模块集成在一起)，找毛绒厂打个像样的样。
    *   **目标**：给 10 个用户试用，验证佩戴舒适度和电池续航。

3.  **阶段三：小批量产 (半年后)**
    *   **投入**：~50k - 100k。
    *   **行动**：开模具，做认证，批量备料。
