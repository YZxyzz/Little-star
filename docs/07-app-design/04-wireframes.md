### 1.2 内容规格 (Content Specs)

| 区域 | 元素 | 数据来源 | 文案/逻辑 |
|------|------|---------|----------|
| **顶部栏** | 孩子切换 | `User.children` | 显示当前孩子昵称，点击下拉切换 |
| | 设备状态 | `Device` | 🟢在线+电量+信号，点击→设备管理 |
| **报告总览** | 一句话总结 | `Report.summary` | AI生成，限30字 |
| | 情绪 | `Report.mood` | 😊/😐/😢 + 文字 |
| | 话题标签 | `Report.topics` | 前3个热门话题 |
| **今日精选** | 动态卡片 | `Conversations` | 最多显示3条，按重要性排序 |
| **互动策略** | 建议卡片 | `Suggestions` | 针对今日事件的育儿建议 |

## 四、关键文案库 (Copywriting Bank)

| 内部术语 | 用户端展示文案 | 备注 |
|---------|--------------|------|
| User | 爸爸/妈妈 | 根据角色动态变化 |
| Device | 小星伴 | 或用户自定义名称 |
| Report | 今日报告 | 避免"监控报告" |
| Suggestion | 互动策略 | 避免"教育指导" |
| Conversation | 对话记录 | 或"精彩瞬间" |
| AI Chat | 问问小星伴 | 强调AI了解孩子 |

## 五、状态规格

### 5.1 空状态

| 页面 | 场景 | 文案 | 操作 |
|------|------|------|------|
| **首页** | 新用户无数据 | "小星伴正在竖起耳朵听~" | [如何开始?] |
| **记录** | 无对话记录 | "还没有收到对话记录哦" | [了解更多] |
| **AI对话** | 首次进入 | "我是小星伴，问我任何关于小明的问题！" | - |

### 5.2 错误状态

| 场景 | 提示文案 | 解决方案 |
|------|---------|---------|
| 断网 | "网络似乎断开了" | 检查网络设置 |
| 设备离线 | "小星伴暂时联系不上" | 检查设备电量和信号 |
| AI响应超时 | "思考中...再等一下" | 自动重试 |
