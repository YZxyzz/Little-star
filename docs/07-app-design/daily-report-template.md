# 每日成长报告模板设计 v4.0

> **设计目标**：让父母深度了解孩子，提炼重点问题，提供可执行的行动指导
>
> **核心理念**：基于孩子"说了什么"来了解孩子，而不是假装知道孩子"做了什么"
>
> **v4.0 更新**：新增三层信息架构、渐进式披露、历史对比、家长反馈闭环等优化

---

## 技术能力边界（重要）

### 我们能做到的 ✅

| 能力 | 数据来源 | 说明 |
|------|---------|------|
| 知道孩子说了什么 | 语音转文字 | 孩子和小星伴的对话内容 |
| 知道什么时候说的 | 录音时间戳 | 对话发生的时间 |
| 分析对话中的情绪 | 语音情绪识别 + 文本分析 | 这段对话时孩子的情绪 |
| 提取孩子提到的事 | LLM内容分析 | 孩子主动说的经历、人物、话题 |

### 我们做不到的 ❌

| 限制 | 原因 |
|------|------|
| 不知道孩子实际行程 | 只有对话录音，不知道孩子在哪 |
| 不知道没说的事 | 孩子不说，我们就不知道 |
| 连续情绪曲线 | 只有离散的对话片段，不是持续监测 |
| 精确时间线 | "09:00语文课"这种只有孩子说了才知道 |

### 设计原则

> **诚实原则**：只呈现我们真正知道的，不假装知道不知道的
>
> **数据来源**：所有内容都来自"孩子主动说的话"，报告要让父母清楚这一点

## 零、三层信息架构（v4.0 新增）

> **核心思想**：日报不是家长了解孩子的唯一方式，而是三层信息体系的中间层。
> 实时提醒负责"即时感知"，日报负责"深度了解"，长期追踪负责"成长趋势"。

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                          三层信息架构                                         │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  第一层：⚡ 实时推送（即时触达）                                              │
│  ├── 🚨 情绪告警：检测到孩子哭泣/极度低落 → 立即推送                         │
│  ├── 💬 孩子金句：孩子说了一句暖心/有趣的话 → 推送给家长                      │
│  └── ❓ 好奇心爆发：孩子连续问了多个同主题问题 → 推一条                       │
│                                                                              │
│  ⏱️ 延迟：秒级 | 频率：每天 0-5 条 | 家长行动：看一眼就好                     │
├──────────────────────────────────────────────────────────────────────────────┤
│  第二层：📋 每日报告（本文档核心）                                            │
│  ├── 今日亮点 + 概览 + 对话记录 + 孩子的世界                                 │
│  ├── 值得关注 + 今晚行动 + 亲子对话建议                                      │
│  └── 家长反馈入口（形成闭环）                                                │
│                                                                              │
│  ⏱️ 每天傍晚 17:00-18:00 推送 | 家长行动：花 3-5 分钟仔细看                  │
├──────────────────────────────────────────────────────────────────────────────┤
│  第三层：📈 成长追踪（周报/月报/里程碑）                                      │
│  ├── 周报：本周情绪趋势、兴趣变化、社交关系演变                               │
│  ├── 月报：月度成长档案、发展里程碑、长期兴趣追踪                             │
│  └── 里程碑：首次主动道歉 / 第一次问哲学问题 / 等等                           │
│                                                                              │
│  ⏱️ 周末/月末 | 家长行动：和伴侣一起看，规划教育方向                          │
└──────────────────────────────────────────────────────────────────────────────┘
```

### 第一层：实时推送设计

#### 金句推送（杀手级功能）

```
┌────────────────────────────────────────┐
│  💬 小明刚才说了一句话：               │
│                                        │
│  "小星伴，你说宇航员在太空            │
│   会不会想妈妈？我如果去太空           │
│   肯定要带着妈妈一起去！"             │
│                                        │
│  ⏰ 17:15 · 😊 开心                    │
│  ❤️ 收藏   💬 看完整对话               │
└────────────────────────────────────────┘
```

**触发规则**：
| 推送类型 | 触发条件 | 频率限制 | 推送延迟 |
|---------|---------|---------|---------|
| 情绪告警 | 情绪检测为"难过""害怕"且持续 >2 分钟 | 每天最多 2 条 | 即时 |
| 孩子金句 | LLM 判定为有趣/暖心/有教育意义的话 | 每天最多 3 条 | 5 分钟内 |
| 好奇心爆发 | 同一话题连续提问 ≥3 个问题 | 每天最多 1 条 | 10 分钟内 |

**为什么金句推送是杀手级功能**：
- 日报是"理性了解孩子"，金句是"感性连接孩子"
- 每天收到一两条来自孩子的暖心话，是家长愿意续费的核心动力
- 参考 Nanit 的"今日最佳瞬间"——是用户粘性最高的功能

## 二、各模块详细设计

### Part 0: ✨ 今日亮点（v4.0 新增）

**目的**：家长打开报告后看到的第一个东西，决定了他们会不会继续往下看

```
┌──────────────────────────────────────────────────────────────────────────────┐
│  ✨ 今日亮点                                                                 │
│                                                                              │
│  "小明今天兴奋地问了5个关于宇宙的问题，                                       │
│   还把精心画的太空画送给了好朋友——虽然                                        │
│   结果不如他期待，但他在对话中逐渐释怀了。"                                   │
│                                                                              │
│  一句话：今天有波折，但孩子展现了自我调节能力 💪                               │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
```

**设计要点**：
- 用叙事性语言而非列表，让家长有"读故事"的感觉
- 一句话总结提炼今天最核心的信息
- 既提到正面（好奇心、自我调节）也提到需要关注的（波折）
- 长度控制在 3-4 行，10 秒读完

- 用孩子自己分享的事来组织信息
- 对比数据让家长了解趋势

- **快速评分**：一键反馈，不增加负担
- **文字反馈**：家长可以补充今晚和孩子聊的结果
- **闭环价值**：家长的反馈可以校正 AI 分析偏差，下次报告更精准
- **报告局限说明**：进一步强化"诚实原则"，坦诚告知报告的边界
- **引导行动**：鼓励家长主动问孩子更多

### 6.2 AI 生成提示词（v4.0）

```
你是一个儿童心理专家和育儿顾问。

以下是孩子今天和小星伴的对话记录（包含时间戳和内容）。
请基于这些对话生成一份每日成长报告。

重要原则：
- 只基于孩子说的内容来分析，不要假设孩子没说的事情
- 情绪分析基于每次对话，不要画连续曲线
- 所有信息都要标注"孩子说"，让父母知道数据来源
- 引用孩子的原话，让父母"听到"孩子的声音
- 与昨天和本周平均数据对比，展示趋势
- 资源推荐必须基于孩子今天的具体对话内容，不要给通用推荐

报告结构：
0. 今日亮点：用3-4行叙事性语言总结今天最核心的信息，附一句话提炼
1. 今日概览：孩子今天和小星伴聊了什么，含历史对比数据
2. 对话记录：按时间排列，每段对话的情绪和内容
3. 孩子的世界：从对话中了解到的（心情/朋友/兴趣/困扰），四个维度分别输出
4. 值得关注：需要父母留意的事（引用原话），附场景化AI入口提示
5. 今晚行动：具体怎么和孩子聊，包括对话示例
6. 亲子对话建议：基于孩子今天具体的提问和兴趣，给出个性化资源推荐
7. 报告说明：基于几次对话生成，坦诚报告边界

语言风格：温暖、专业、不说教
```

### 6.3 实时推送触发提示词

```
你是一个儿童心理分析专家。

分析以下刚刚发生的孩子对话片段，判断是否需要即时推送通知给家长。

推送类型判断：
1. 情绪告警：孩子表达了强烈的负面情绪（难过、害怕、愤怒）且持续时间超过2分钟
2. 孩子金句：孩子说了一句特别暖心、有趣、有教育意义的话
3. 好奇心爆发：孩子在同一话题连续提问3个以上问题

如果判断需要推送，输出：
- 推送类型
- 推送标题（10字以内）
- 推送内容（引用孩子原话，30字以内）
- 情绪标签

如果不需要推送，输出："不需要推送"
```

## 七、设计原则

1. **数据诚实**：只呈现我们真正知道的，不假装知道不知道的
2. **孩子视角**：大量引用孩子原话，让父母"听到"孩子的声音
3. **来源透明**：所有内容标注来源，让父母知道信息从哪来
4. **问题前置**：重要的问题要醒目，不让父母错过
5. **可执行性**：建议具体到"怎么说、怎么做"
6. **教育父母**：不只告诉问题，更教父母如何处理
7. **正向引导**：不只关注问题，也发现孩子的闪光点
8. **渐进披露**：三级展开，无论看多少都能获得最有价值的信息（v4.0）
9. **闭环反馈**：家长反馈→改进报告→更精准的下一次（v4.0）
10. **个性化推荐**：基于当天对话内容，不给通用推荐（v4.0）
