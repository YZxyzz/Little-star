## 一、应用状态定义

### 1.1 用户状态流转

```mermaid
stateDiagram-v2
    [*] --> 未登录状态
    未登录状态 --> 已登录未绑定: 登录/注册成功
    已登录未绑定 --> 设备绑定中: 开始绑定
    设备绑定中 --> 激活成功: 绑定完成
    激活成功 --> 正常使用: 动画结束
    正常使用 --> 已登录未绑定: 解绑设备
    正常使用 --> 未登录状态: 退出登录
```

### 1.2 状态说明

| 状态 | 进入条件 | 可见内容 | 核心操作 |
|------|---------|---------|---------|
| **未登录** | 首次打开/退出登录 | 欢迎页、登录页 | 登录/注册 |
| **已登录未绑定** | 登录成功但无设备 | 绑定引导页 | 绑定设备 |
| **设备绑定中** | 开始绑定流程 | 绑定向导页 | 完成绑定步骤 |
| **激活成功** | 设备绑定成功 | 激活成功动画 | 自动跳转/点击进入 |
| **正常使用** | 有已绑定设备 | 完整3 Tab界面 | 全部功能 |

## 二、全局导航设计

### 2.1 导航模式
采用 **底部标签栏 (Bottom Tab Bar)** 作为一级导航，辅以 **顶部导航栏 (Top Navigation Bar)** 进行层级深入。

**重要**：底部导航仅在「正常使用」状态下显示，未登录和绑定流程中不显示。

### 2.2 底部标签定义 (Tab Bar) - 3 Tab架构

| 图标 | 名称 | 核心功能 | 选中态说明 |
|------|------|---------|-----------|
| 🏠 | **首页 (Home)** | 设备状态、报告总览、今日精选、互动策略 | 快速了解孩子今天怎么了 |
| 💬 | **记录 (Records)** | 个人记忆图谱、新动态推送、AI对话入口 | 深度了解孩子的内心世界 |
| 👤 | **我的 (Mine)** | 孩子档案、设备管理、账号设置 | 低频但必要的配置项 |

## 四、页面详细结构

### 4.1 未登录状态页面

#### P-00 启动画面 (Splash Screen)
*   **时长**：1.5-2秒
*   **内容**：品牌Logo + 产品名称 "小星伴"
*   **动画**：Logo呼吸闪烁效果
*   **背景**：渐变色（品牌主色调）

#### P-01 欢迎页 (Welcome Page)
*   **定位**：产品价值展示，引导用户登录
*   **内容**：
    *   品牌IP形象大图
    *   产品Slogan："读懂孩子，陪伴成长"
    *   核心价值点（3个轮播卡片）
        *   "听懂孩子的心声"
        *   "科学育儿建议"
        *   "记录成长轨迹"
*   **操作**：
    *   [开始使用] → 登录页
    *   [了解更多] → 产品介绍页（可选）

#### P-02 登录/注册页 (Login Page)
*   **登录方式**：
    *   手机号 + 验证码（主要）
    *   微信一键登录（快捷）
*   **表单**：
    *   手机号输入框
    *   验证码输入框 + 获取按钮（60s倒计时）
*   **协议**：
    *   勾选框 + 《用户协议》《隐私政策》链接
*   **按钮**：[登录/注册] （手机号自动判断新老用户）

---

### 4.2 设备绑定流程页面

#### P-10 绑定引导页 (Binding Guide)
*   **定位**：引导新用户开始绑定设备
*   **内容**：
    *   IP形象插图（小星伴等待连接）
    *   标题："让我们开始吧！"
    *   说明文案："连接小星伴，开启孩子成长记录之旅"
    *   设备准备检查清单：
        *   ✅ 确保设备已充电
        *   ✅ 将设备放在身边
*   **操作**：
    *   [开始绑定] → 绑定向导 Step1
    *   [稍后绑定] → 返回（但功能受限提示）

#### P-11 绑定向导 - Step1: 创建孩子档案
*   **进度指示**：Step 1/5
*   **标题**："告诉我们关于孩子的信息"
*   **表单**：
    *   孩子昵称（必填）- 用于AI称呼
    *   孩子姓名（选填）- 正式档案
    *   生日（必填）- 用于年龄计算
    *   性别（必填）- 用于AI语境
    *   头像（选填）- 可上传或选择预设
*   **说明**："这些信息帮助AI更好地理解您的孩子"
*   **操作**：[下一步] / [返回]

#### P-12 绑定向导 - Step2: 准备设备
*   **进度指示**：Step 2/5
*   **标题**："准备您的小星伴"
*   **内容**：
    *   设备示意图/动画
    *   操作指引：
        1. 找到设备底部的电源开关
        2. 长按开关 3 秒，直到指示灯亮起
        3. 确保设备在手机附近（1米内）
*   **动画**：循环播放开机操作动画
*   **提示**："设备指示灯闪烁蓝色表示已就绪"
*   **操作**：[设备已就绪，开始扫描] / [返回]

#### P-13 绑定向导 - Step3: 扫描配对
*   **进度指示**：Step 3/5
*   **标题**："正在搜索小星伴..."
*   **内容**：
    *   扫描动画（雷达波扩散效果）
    *   搜索状态文案
*   **发现设备后**：
    *   弹出设备卡片："发现设备：小星伴-XXXX"
    *   设备信息预览（型号、信号强度）
    *   [连接此设备] 按钮
*   **超时处理**（30秒）：
    *   "未找到设备" 提示
    *   [重新扫描] / [查看帮助]
*   **帮助入口**：常见问题FAQ

#### P-14 绑定向导 - Step4: 连接中
*   **进度指示**：Step 4/5
*   **标题**："正在连接..."
*   **内容**：
    *   连接进度动画（小星伴IP与手机握手）
    *   状态文案动态变化：
        1. "正在建立连接..."
        2. "正在验证设备..."
        3. "正在同步信息..."
        4. "连接成功！"
*   **预计时长**：5-15秒
*   **失败处理**：
    *   "连接失败，请重试"
    *   [重试] / [联系客服]

#### P-15 绑定向导 - Step5: 激活服务
> **设计原则**：绑定流程中不展示任何价格信息

*   **进度指示**：Step 5/5
*   **标题**："即将完成！"
*   **内容**：
    *   设备状态确认：
        *   ✓ 设备已连接
        *   ✓ 信息已同步
    *   功能说明（无价格）：
        *   "全天候陪伴对话"
        *   "自动记录成长点滴"
        *   "生成每日成长报告"
*   **操作**：
    *   [激活小星伴] → 激活成功动画
    *   激活过程约3-5秒
*   **注意**：价格信息仅在"我的→会员中心"展示

#### P-16 激活成功动画页 ✨
*   **定位**：庆祝时刻，增强用户成就感
*   **动画设计**（炫酷动效）：
    1. **星星绽放**（0-1s）：
       - 屏幕中心出现小星伴IP
       - 周围星星粒子爆发散开
    2. **光环扩散**（1-2s）：
       - IP形象发出光环波纹
       - 彩色渐变光效流动
    3. **成功标识**（2-3s）：
       - "激活成功！" 文字弹出
       - 金色勋章/徽章效果
    4. **祝福语**（3-4s）：
       - "小星伴已准备好"
       - "开启孩子成长记录之旅"
    5. **引导按钮**（4s后）：
       - [开始探索] 按钮渐显
       - 或 5秒后自动跳转首页
*   **音效**：轻快的成功音效（可选）
*   **背景**：渐变星空/彩带飘落效果

### 4.4 💬 记录 (Records) - 个人记忆图谱系统

> **设计目标**：构建孩子的"个人认知画像"，让家长从6个维度深度了解孩子的内心世界。
>
> **重大变更 (2026-02-09)**：从话题索引+对话流重构为记忆图谱系统。
>
> **数据架构**：双模数据融合（后端逻辑，前端不暴露数据来源）
> - **主动对话**（按键触发）→ 后端标记为高优先，影响排序和权重
> - **被动监听**（始终在线）→ 后端作为背景佐证，AI提取后融合呈现

#### P-30 记录主页（记忆图谱 v2）
*   **顶部**：搜索入口
*   **新动态区域**（时间线）：
    *   AI 自动推送的新发现卡片
    *   4级推送：🔴 即时告警 / 🟡 当日高亮 / 🟢 周更趋势 / 💎 惊喜成就
    *   卡片内容由AI融合双模数据后呈现，不向家长暴露数据来源
*   **记忆图谱区域**（6个入口卡片）：
    *   👥 人物图谱（社交世界）- 红点提示新动态
    *   💛 情绪档案（情绪模式）
    *   🔭 兴趣图谱（兴趣演变）- 红点提示新动态
    *   🌱 成长足迹（关键里程碑）- 红点提示新动态
    *   ⚡ 正在经历（持续挑战）
    *   🌍 小小世界观（孩子的理论）
*   **底部固定**：AI对话入口栏

#### P-31 人物图谱详情页
*   **人物列表**：按互动频率排序
*   **每个人物卡片**：
    *   名字 + 关系标签（好朋友/同学/老师...）
    *   关系状态指示：💚 亲密 / ⚠️ 紧张 / ✨ 新认识
    *   提及频率 + 趋势
    *   关键事件摘要
    *   日常互动细节
*   **声纹识别说明**：被动监听模式下自动建档

#### P-31b 情绪档案详情页
*   **情绪模式总结**（AI叙事，非数据标签）
*   **本周情绪趋势图**
*   **关键发现**：什么场景容易触发焦虑？什么时候最放松？
*   **情绪稳定性评估**（周/月维度）

#### P-31c 兴趣图谱详情页
*   **兴趣列表**：按热度+深度排序
*   **每个兴趣**：
    *   演变路径：恐龙 → 古生物 → "为什么会灭绝"

#### P-31d 成长足迹详情页

### 5.1 首次启动完整流程

```mermaid
sequenceDiagram
    participant User as 新用户

    Note over User,Cloud: 登录/注册
    User->>App: 输入手机号
    App->>Cloud: 发送验证码
    Cloud-->>User: 短信验证码
    User->>App: 输入验证码
    App->>Cloud: 验证并创建/登录账户
    Cloud-->>App: 登录成功
    
    Note over User,Cloud: 检查设备状态
    App->>Cloud: 查询绑定设备
    Cloud-->>App: 返回：无设备
    App-->>User: 展示绑定引导页
    
    Note over User,Cloud: 设备绑定流程

    App-->>User: Step2: 准备设备
    User->>Device: 长按3秒开机
    Device-->>User: 蓝灯闪烁
    User->>App: 点击"开始扫描"
    App-->>User: Step3: 扫描中...
    App->>Device: BLE扫描

    Cloud-->>App: 绑定确认
    App-->>User: Step5: 激活流量

    App->>Cloud: 激活蜂窝流量
```

### 5.2 设备绑定详细流程 (BLE + Cloud)

```mermaid
sequenceDiagram
    participant User as 用户
    participant App as APP
    participant BLE as 蓝牙模块
    participant Device as 小星伴
    participant Cloud as 云端

    Note over User,Cloud: Step3: 扫描配对
    User->>App: 点击"开始扫描"
    App->>BLE: 启动BLE扫描
    BLE-->>App: 扫描中...
    
    loop 扫描设备 (最多30秒)
        BLE->>Device: 扫描BLE广播
        Device-->>BLE: 广播设备信息
        BLE-->>App: 发现设备(RSSI判定距离)
    end
    
    App-->>User: 弹出设备卡片"小星伴-XXXX"
    User->>App: 点击"连接此设备"
    
    Note over User,Cloud: Step4: 建立连接
    App->>BLE: 发起BLE连接
    BLE->>Device: 连接请求
    Device-->>BLE: 连接确认
    BLE-->>App: 已连接
    
    App->>Device: 读取设备信息(型号/固件版本)
    Device-->>App: 返回设备信息
    
    App->>Cloud: 注册设备(设备ID+用户ID+孩子ID)
    Cloud-->>App: 绑定成功
    
    App->>Device: 下发配置(孩子信息)
    Device-->>App: 配置完成
    
    Note over User,Cloud: Step5: 激活流量
    App->>Cloud: 请求激活流量
    Cloud->>Device: 4G模块激活指令
    Device-->>Cloud: SIM卡注册成功
    Cloud-->>App: 流量激活完成
    
    Note over User,Cloud: 🎉 完成
    App-->>User: 播放激活成功动画
    App->>BLE: 断开BLE连接(后续通过4G通信)
```

### 5.3 日常使用流程

```mermaid
sequenceDiagram
    participant User as 家长
    participant App as APP首页
    participant Detail as 报告详情页
    participant AI as AI对话
    participant Action as 现实行动

    User->>App: 打开 APP
    App-->>User: 展示首页(设备状态+报告总览)

        App->>Detail: 跳转报告详情页
        User->>Detail: 阅读完整报告
        User->>Detail: 点击展开互动建议
        Detail-->>User: 展示对话模板
        User->>Detail: 复制话术

    else AI问答

        User->>AI: 点击AI对话入口
        User->>AI: 提问"小明最近对什么感兴趣？"

        User->>Action: 根据AI建议和孩子互动
    end
```

### 5.4 解绑与重新绑定流程

```mermaid
sequenceDiagram

    participant App as APP

    User->>App: 点击"开始绑定"

```
