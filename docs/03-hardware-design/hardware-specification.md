# 可穿戴设备硬件规格说明

## 一、设计目标

### 1.1 核心设计原则

1. **安全第一**：所有材料符合儿童产品安全标准
2. **轻量便携**：让孩子愿意全天佩戴
3. **持久续航**：满足一整天使用需求
4. **可爱吸引**：孩子主动愿意佩戴
5. **耐用防护**：适应儿童使用场景

### 1.2 目标规格概览

> ⚠️ **重要更新 (2025-02-05)**：续航要求提升至48小时，支持独立网络。

| 参数 | 目标值 | 说明 |
|------|--------|------|
| 总重量 | <50g | 毛绒外壳~20g + 硬件模块~30g |
| 硬件模块尺寸 | 40×40×18mm | 容纳更大电池 |
| **续航** | **>48小时** | **持续录音模式，2天充一次电** |
| 充电时间 | <3小时 | USB-C快充 |
| **网络** | **内置蜂窝流量卡** | **不依赖手机/WiFi，直连云端** |
| 防水等级 | 硬件模块IP54 | 毛绒外壳可拆卸清洗 |
| 工作温度 | -10°C ~ 45°C | - |
| 存储容量 | 16GB | 约500小时录音（离线缓存） |
| 振动反馈 | 支持 | LRA线性马达 |
| **实时传输** | **支持** | **蜂窝网络实时上传音频** |

### 2.2 核心组件选型

#### 主控芯片

| 候选 | 型号 | 特点 | 推荐度 |
|------|------|------|--------|
| **方案A** | **ESP32-S3 + 4G模组** | BLE配对 + 蜂窝数据传输 | **★★★★★** |
| 方案B | nRF9160 | 集成LTE-M/NB-IoT, 低功耗 | ★★★★☆ |
| 方案C | 高通MDM9206 | 全功能LTE Cat.M1 | ★★★☆☆ |

**推荐**：ESP32-S3 + 移远EC20/SIM7600 4G模组
- ESP32-S3负责音频处理、BLE配对
- 4G模组负责蜂窝数据传输（Cat.1/Cat.4）
- 内置eSIM或Nano SIM卡槽

#### 麦克风

| 候选 | 型号 | 特点 | 推荐度 |
|------|------|------|--------|
| 方案A | Knowles SPH0645LM4H | 数字输出, 高信噪比 | ★★★★★ |
| 方案B | InvenSense ICS-43434 | 低功耗, 高性能 | ★★★★☆ |

**推荐**：双麦克风阵列（降噪需要）

#### 存储

| 候选 | 容量 | 特点 | 推荐度 |
|------|------|------|--------|
| 方案A | 8GB NAND Flash | 大容量, 成本适中 | ★★★★★ |
| 方案B | 4GB NAND Flash | 成本更低 | ★★★☆☆ |

**推荐**：8GB（可存储约250小时音频）

#### 电池

> ⚠️ **更新**：为满足48小时续航需求，电池容量大幅提升。

| 候选 | 容量 | 特点 | 推荐度 |
|------|------|------|--------|
| **方案A** | **1200mAh Li-Po** | **满足48h续航目标** | **★★★★★** |
| 方案B | 1000mAh Li-Po | 约40小时续航 | ★★★★☆ |
| 方案C | 1500mAh Li-Po | 续航更长但更重 | ★★★☆☆ |

**推荐**：1200mAh（约48-52小时续航，2天充一次电）

**续航计算**：
- 持续录音功耗：约20mA
- 蜂窝网络间歇传输：平均8mA
- 总平均功耗：约28mA
- 续航时间：1200mAh ÷ 28mA ≈ 43小时（实际使用约48小时，因有静默期）

#### 振动马达

| 候选 | 类型 | 特点 | 推荐度 |
|------|------|------|--------|
| 方案A | ERM（偏心转子）| 成本低，振感强 | ★★★★☆ |
| 方案B | LRA（线性谐振）| 响应快，可调节 | ★★★★★ |

**推荐**：LRA线性马达（响应速度快，触感细腻）

### 3.4 材料选择

> ⚠️ **重要更新 (2026-02-08)**：外壳材质改为硅胶方案，更适合儿童使用场景

#### 主要方案：硅胶外壳（推荐）

| 部件 | 材料 | 要求 | 优势 |
|------|------|------|------|
| **硅胶外壳** | 医用级液态硅胶 | 食品接触级、邵氏硬度30-40A | 防水、易清洁、耐用 |
| **眼睛** | 一体成型硅胶眼睛 | 无脱落风险 | 安全、可水洗 |
| **硬件壳体** | 食品级ABS | 保护内部电子元件 | - |
| **磁吸背片** | 硅胶 + 钕磁铁 | 适中磁力、安全包裹 | - |
| **别针扣** | 不锈钢 + 塑料安全扣 | 防刺伤设计 | - |
| **挂绳** | 棉质编织绳 | 安全断裂设计 | - |
| **充电线** | TPE软胶线 | 柔韧耐用 | - |

**硅胶外壳核心优势：**
- ✅ **防水等级 IP67**：可直接水洗，无需拆卸硬件
- ✅ **易清洁**：湿布擦拭或水冲即可，适合儿童使用
- ✅ **耐用性强**：不怕摔、不怕脏、不易损坏
- ✅ **卫生安全**：医用级材质，可煮沸消毒
- ✅ **可更换造型**：不同卡通形象套壳（兔子、海豚、小熊等）

#### 可选方案：毛绒外套（配件）

作为可选配件提供毛绒外套，满足特定场景需求：

| 部件 | 材料 | 使用场景 |
|------|------|---------|
| **毛绒外套** | 短绒面料（聚酯纤维） | 冬季、睡前陪伴 |
| **填充物** | PP棉 | 增加温暖触感 |

**使用方式：**
- 套在硅胶外壳外层
- 脏了可拆卸单独清洗
- 不影响硬件防水性能

#### 硅胶套设计规范

**可更换卡通造型系列：**

| 系列 | 造型 | 目标用户 | 特点 |
|------|------|---------|------|
| **基础款** | 星星 ★ | 全部 | 品牌标准形象 |
| **动物系列** | 兔子 🐰 | 喜欢可爱风格 | 长耳朵识别度高 |
| **海洋系列** | 海豚 🐬 | 喜欢海洋动物 | 流线型设计 |
| **森林系列** | 小熊 🐻 / 狐狸 🦊 | 喜欢森林动物 | 圆润可爱 |
| **节日款** | 圣诞、春节等 | 节日限定 | 季节性周边 |

**硅胶套材质规格：**
- 材质：医用级液态硅胶（LSR）
- 硬度：邵氏硬度 30-40A（柔软舒适）
- 厚度：3-5mm
- 安全标准：
  - 符合 GB 4806.11-2016 食品接触用橡胶材料标准
  - 符合 FDA 21 CFR 177.2600 食品级标准
  - 无双酚A、无邻苯二甲酸酯、无重金属
- 耐温范围：-40°C ~ 200°C
- 防水等级：IP67（可水洗、可煮沸消毒）

**清洁维护：**
- 日常清洁：湿布擦拭
- 深度清洁：温水冲洗 + 中性清洁剂
- 消毒方式：煮沸消毒（5-10分钟）或紫外线消毒

### 3.6 安全设计

1. **无小零件**：眼睛牢固嵌入，所有配件固定连接
2. **安全挂绳**：超过2kg拉力自动断开，防勒颈
3. **磁铁安全**：磁铁完全包裹在硅胶内，防止误吞
4. **圆角设计**：硬件模块所有边缘圆滑处理
5. **阻燃材料**：毛绒外壳符合阻燃标准
6. **可水洗设计**：取出硬件模块后，毛绒外壳可机洗
7. **材料安全**：
   - 符合GB 6675玩具安全标准
   - 符合EN 71欧盟玩具安全标准
   - 无邻苯二甲酸酯
   - 无双酚A
   - 无重金属

### 4.7 情感化反馈设计

> **设计理念**：让硬件"活"起来，像一个有生命的小伙伴

#### 情感振动模式

| 情感状态 | 振动模式 | 触发场景 | 拟人化感受 |
|---------|---------|---------|-----------|

| **思考** | 不规则轻振（● ● ... ●） | AI正在处理回复 | 挠头想问题 |

| **撒娇** | 连续轻柔振（～～～） | 很久没人理它 | 想要关注 |
| **困了** | 渐弱振动（●...●.....●......） | 电量低 | 打哈欠 |

| **需要充电** | "电量不足10%" | "好困好困...眼睛都睁不开了..." |
| **充电中** | "正在充电" | "啊～好舒服，谢谢你～" |
| **充满电** | "充电完成" | "我睡醒啦！精神满满！" |
| **配对成功** | "蓝牙已连接" | "耶！我们连上啦～" |
| **断开连接** | "连接已断开" | "等等，你去哪儿了？" |
| **网络异常** | "网络错误" | "唔...我好像有点迷路了" |
| **长时间未使用** | （无提示） | "好无聊啊...有人在吗？" |

#### 生命感设计原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **有呼吸感** | 即使静止也有微弱的存在感 | 待机时LED缓慢呼吸 |
| **会累会困** | 电量低时行为变化 | 反应变慢、声音变软 |
| **有情绪** | 对互动有情感反应 | 被冷落时会"撒娇" |
| **不完美** | 偶尔有小状况 | "我刚才走神了，再说一遍？" |
| **会成长** | 随使用时间变化 | 记住孩子的喜好和习惯 |

### 7.2 核心功能

1. **智能录音**
   - 语音活动检测（VAD）
   - 自动开始/停止录音
   - 环境噪音过滤

2. **数据同步**
   - 蜂窝网络实时同步（主要）
   - BLE备用同步（网络异常时）
   - 增量同步机制
   - 离线缓存 + 网络恢复后自动上传

3. **低功耗管理**
   - 深度睡眠模式
   - 动态频率调节
   - 智能唤醒

4. **OTA升级**
   - 安全固件升级
   - 断点续传
   - 回滚保护
